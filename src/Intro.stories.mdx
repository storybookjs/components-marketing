# Get started

UI components for the Storybook marketing, docs, addons, blog and showcase sites.

## Usage

Components are written in React, and its stories are written in [Component Story Format](https://storybook.js.org/docs/react/api/csf#gatsby-focus-wrapper). It requires Storybook version 6.5 and up.

**Install**

```
npm install --save @storybook/components-marketing
```

**Import components you want into your UI**

```bash
import { Eyebrow, GithubButton } from '@storybook/components-marketing';
```

and use them like so

```jsx
const example = () => (
  <div>
    <Eyebrow
      label="Storybook Lazy Compilation for Webpack"
      link="https://storybook.js.org/blog/storybook-lazy-compilation-for-webpack/"
    />
    <GithubButton />
  </div>
);
```

### **Run and develop locally**

Clone the [Components Marketing GitHub project](https://github.com/storybookjs/components-marketing) then start Storybook.

```bash
# install dependencies
yarn
# start storybook
yarn run storybook
```
